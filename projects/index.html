Skills

To classify my skills, I've come up with a method to determine my level of expertise in each of the technologies I have used throughout my professional career, which consists of a 1 to 5 star rating with the following meaning:


Legend
5 I consider myself kind of an expert with this technology. I can always improve, but I feel very comfortable using this one in a project.
4 I've used this technology several times and I can do advanced stuff with it.
3 I know some intermediate things about this technology, so I could manage a little project myself.
2 I've used this technology, but I only know basic features about it.
1 I've started to learn this technology, so I won't be experienced with that.
