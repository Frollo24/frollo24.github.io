Who am I?

My name is Adrian, I'm a 23 year old videogame and 3D rendering enthusiast. I have a degree in Videogame Design and Development at at Rey Juan Carlos University, Spain. I'm also finishing a master degree in Computer Graphics, Games and Virtual Reality at the same university.

My main interests are game development, engine development and computer graphics. For my Bachelor Thesis I've developed a render engine using OpenGL as the main API with the ability of rendering PBR materials and for my Master Thesis I'm developing a neural network that predicts the global illumination of a scene.

Computer Graphics

As I mentioned before, I'm a computer graphics enthusiast. Although my BT render engine is done with OpenGL as the main API, I have also developed some render engines using Vulkan as the main API. Furthermore, I have some experience with Mitsuba 3 having used it as the pathtracer for data generation in my MT project.
